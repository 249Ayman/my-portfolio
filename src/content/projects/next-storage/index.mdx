A functional clone of Google Drive designed to demonstrate high-performance file handling and serverless architecture. It allows users to securely upload, organize, and retrieve files using Cloudflare's edge network.

## Key Features

- **Scalable Storage**: Utilizes Cloudflare R2 for zero-egress fee, S3-compatible object storage.
- **Serverless Backend**: Entire application logic runs on Cloudflare Workers for global low latency.
- **File Management**: Supports drag-and-drop uploads, folder creation, and file organization.
- **Efficient Delivery**: Assets are served directly from the edge, ensuring rapid download speeds.

## Technologies Used

- Next.js
- Cloudflare R2 (Object Storage)
- Cloudflare Workers
- Tailwind CSS
- TypeScript
