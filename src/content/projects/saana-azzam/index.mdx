A high-performance personal portfolio website for Saana Azzam, showcasing her expertise as an international speaker, author, and performance coach. The site is engineered to deliver rapid load times and a seamless user experience.

## Key Features

- **Blazing Fast Performance**: Built with Astro's island architecture to minimize JavaScript and maximize speed.
- **Global Delivery**: Deployed on Cloudflare Pages for low-latency access worldwide.
- **Media Showcase**: Optimized video and image galleries highlighting speaking engagements.
- **SEO Optimized**: Structure and performance tuned for high search engine visibility.

## Technologies Used

- Astro
- Cloudflare Pages
- React (for interactive islands)
- Tailwind CSS
