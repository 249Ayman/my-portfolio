Side project I shipped on Cloudflare Workers: help people see how bots read their resume before a human does.

## What it does

- **Free ATS-style scan** on upload (PDF / DOCX, up to the advertised size limit)
- **AI review** for wording, impact, and structure—not just keyword stuffing
- **Builder + templates** so you can fix issues and export without fighting Word

## Stack

- Next.js on Cloudflare Workers
- File upload + analysis pipeline tuned for hiring workflows in 2026
